Generator for the textual representation of class member names.

## class_namer

Import into your pubspec `dev_dependencies` section.

```yaml
dev_dependencies:
  flutter_test:
    sdk: flutter
  class_namer:
    git:
      url: https://git.urancompany.com/v.novykov/class_namer.git
      path: packages/class_namer
```
